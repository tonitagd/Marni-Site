I/O Fanuc management module

For CNC FANUC machines

Purpose

I/O module FSI is developed for CNC FANUC machines.

It acts as transition between input/output FANUC board and the screw terminals for cable attachment, which makes convenient connection between control and machine. The I/O module also frees space in the el. closet.

Each FANUC module has 24 inputs and 16 outputs, which represent the inputs and outputs of a FANUC system. Each and every module has LED indication, power supplied by 24V DC.

Dimensions

length 150mm, width 75mm.
